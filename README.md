# prathamesh7pute Blog

This is a personal blog built with Next.js and MDX. It uses Tailwind CSS for styling and `react-syntax-highlighter` for code blocks.

Quick start

1. Install dependencies

   ```bash
   npm install
   ```

2. Run the dev server

   ```bash
   npm run dev
   ```

3. Open http://localhost:3000

Build for production

```bash
npm run build
npm start
```
